# QV Training Library

A modern web application for call agent training and management. Features agent and manager dashboards, rebuttal management, transitions, call process, and training document uploads.

## Tech Stack
- React (TypeScript)
- Node.js/Express
- MongoDB
- Blue & White Fortune 500 style UI

## Features
- Agent dashboard: Rebuttals, Transitions, Call Process
- Manager dashboard: Content editor, client dropdown, live updates
- Upload Word docs from Google/Microsoft
- Persistent storage

## Setup
1. Install dependencies in both `client` and `server` folders
2. Start backend: `npm run dev` in `server`
3. Start frontend: `npm start` in `client`
